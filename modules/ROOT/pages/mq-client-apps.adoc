= クライアントアプリケーションの設定
ifndef::env-site,env-github[]
include::_attributes.adoc[]
endif::[]
:keywords: MQ, クライアント, アプリケーション, アプリケーション

Anypoint MQ クライアントアプリケーションでは、アプリケーションを MQ に登録できます。登録すると、MQ に
アプリケーションのクライアント ID とクライアントシークレットがリストされます。この情報は Anypoint Studio で Anypoint MQ コネクタを設定するために必要です。

*注意:* 登録時にアプリケーションを閉じる必要はありません。

アプリケーションを登録する手順は、次のとおりです。

. Anypoint Platform で、*[MQ]* をクリックします。
. 左のナビゲーションバーから *[Client Apps (クライアントアプリケーション)]* をクリックします。
+
image::mq-client-apps-in-nav.png[width="177"]
+
. 青い丸のプラスをクリックします。
+
image::mq-blue-plus-circle.png[タスクバーの下の約 3 分の 2 右側にあるクライアントアプリケーションの作成用の丸ボタン]
+
. クライアントアプリケーションの名前を指定します。
+
image::mq-create-client-app-window.png[]
+
. *[Save Changes (変更を保存)]* をクリックします。
. クライアントアプリケーションエントリをクリックし、クライアント ID とシークレットの値を表示します。
+
image::mq-client-apps-detail2.png[]
+
. ID またはシークレットの [Copy (コピー)] ボタンをクリックし、値をコンピュータのクリップボードにコピーします。

*注意*: Chrome ブラウザを使用していて [Copy (コピー)] ボタンが機能しない場合は、*[設定]* > *[詳細設定を表示]* > *[プライバシー]* > *[コンテンツの設定]* > *[プラグイン]* > *[重要なプラグインコンテンツを検出して実行する]* をクリックします。
